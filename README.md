📌 Project Title
Personal Portfolio Website
📖 Description
This is a personal portfolio website developed to showcase my skills, projects, and achievements in web development. The website provides a clean and modern interface where visitors can learn more about me, explore my work, and get in touch easily.
The portfolio highlights my journey as a computer engineering student and demonstrates my ability to design and build responsive web applications. It includes sections such as an introduction, project showcase, and contact details, making it a complete professional profile.
The main goal of this project is to present my technical skills in a practical way and create an online presence that can be shared with recruiters and during internship evaluations.
